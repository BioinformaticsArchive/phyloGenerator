Mac OS

  1. Check to see if you've got NumPy installed. From Terminal, type 'python' and hit enter. Then type 'import scipy' and hit enter. Now quit python by typing 'quit()'. If you don't get an error message (there may be a delay), skip to the BioPython step below. While I give brief instructions below, more thorough instructions can be found here.
  2. Download the latest version.
  3. Go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
  4. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
  5. Type 'python setup.py build' in Terminal and press enter.
  6. Type 'sudo python setup.py install' in Terminal and press enter.

phyloGenerator

Windows

Coming soon...

  1. Download a zipped up copy of the package, go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
  2. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
  3. Make the script 'executable' by typing 'chmod 777 phyloGenerator.py'
  4. You're done! Now read the walkthrough to see how to use phyloGenerator!
