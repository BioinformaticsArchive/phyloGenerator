Installing

Installing phyloGenerator is really just about setting up its dependencies: Python, SciPy, BioPython, and the phylogenetics tools you'll be using. It may seem like a lot to do, but really you just need to click a few buttons every five minutes and the installer scripts will do all the hard work for you. Trust e, it's easier than it looks!

You can easily jump to the Mac, Windows, and Linux instructions below.

MacOS

 For this, you need to open up 'Terminal'. It's in your 'Applications' folder, inside the 'Utilities' folder - you can open it from the dock.
  1. Python
    1. In Terminal, type 'python -v' and hit enter. If the version number is greater than '2.6.0', but does not start with a 3, skip to the Scipy step below.
    2. If it's not, download the latest version of Python 2 - do not download any Python version starting with a 3.
  2. NumPy
    1. Check to see if you've got NumPy installed. From Terminal, type 'python' and hit enter. Then type 'import scipy' and hit enter. Now quit python by typing 'quit()'. If you don't get an error message (there may be a delay), skip to the BioPython step below. While I give brief instructions below, more thorough instructions can be found here.
    2. Download the latest version.
    3. Go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
    4. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
    5. Type 'python setup.py build' in Terminal and press enter.
    6. Type 'sudo python setup.py install' in Terminal and press enter.
  3. SciPy
    1. Check to see if you've got SciPy installed. From Terminal, type 'python' and hit enter. Then type 'import scipy' and hit enter. Now quit python by typing 'quit()'. If you don't get an error message (there may be a delay), skip to the BioPython step below. While I give brief instructions below, more thorough instructions can be found here.
    2. Download the latest version.
    3. Go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
    4. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
    5. Type 'python setup.py build' in Terminal and press enter.
    6. Type 'sudo python setup.py install' in Terminal and press enter. Enter your password and press enter.
  4. BioPython
    1. You likely won't have BioPython installed without knowing about it, but you can check by opening Python ('python', hit enter), trying to load a module from it ('from Bio import SeqIO', hit enter), and then quitting Python ('quit()', hit enter). If you don't get an error message (there may be a delay), skip to the step below. While I give brief instructions below, more thorough instructions can be found here.
    2. Download the latest version.
    3. Go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
    4. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
    5. Type 'python setup.py build' in Terminal and press enter.
    6. Type 'python setup.py install' in Terminal and press enter.
  5. Installing phyloGenerator
    1. Download a zipped up copy of the package, go to the directory you downloaded it to in Finder, and then unzip it by double-clicking on it.
    2. Move into the unziped folder in Terminal, by typing 'cd ' and then dragging the folder icon in your Terminal window.
    3. Make the script 'executable' by typing 'chmod 777 phyloGenerator.py'
    4. You're done! Now read the tutorial to see how to use phyloGenerator!
