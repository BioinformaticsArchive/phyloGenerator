One-Click Install

Windows

Unzip this file, then double-click on 'phyloGenerator.exe'.

Have a quick look through the 'README.txt' file, and check out the 'Demos' folder for examples of how to get it working from the command prompt.

Tested on Windows 7. Someone reported a problem on Windows XP, but I've been unable to reproduce it. Please let me know if you have problems!

Mac OS

Unzip this file, then run './phyloGenerator.app/Contents/MacOS/phyloGenerator' from Terminal when in the directory you unzipped it all into.

Have a quick look through the 'README.txt' file for more details on how this works, and check out the 'Demos' folder for examples of how to get it working from the command prompt.

Tested on Mac OS 10.6 and 10.7 (Snow Leopard and Lion)

Compiling from Source

Overview

There're two main steps to this: setting up Python and the packages phyloGenerator uses, and setting up all the phylogenetics programs that phyloGenerator uses. It's not technically challenging, but it is fiddly. Sorry!

Once everything's set up, just run the file 'phyloGenerator.py'. That's it!

Python and its packages

  1. Python. Grab the latest version of Python 2.7. Do not download any Python version starting with a 3.
  2. NumPy. Grab it from here - make sure to get the version for Python 2.7, not 2.6 (SourceForge tends to take you there by default.)
  3. SciPy. Grab it from here - make sure to get the version for Python 2.7, not 2.6 (SourceForge tends to take you there by default.)
  4. BioPython. Grab it from here.

Phylogenetics Software

You don't need all of these programs (...only the ones you're going to use...) but the complete list is below. Once you've compiled them, put them all inside a folder named 'requires' in the same folder as 'phyloGenerator.py'. Some of the binaries need to be named something specific - this name is given after the name of the program (which is a link to its location).

  • RAxML - raxml
  • BEAST - beast. You need to alter the script in the 'bin' folder of the BEAST distribution, as I've done in the source code (the file 'beast').
  • TreeAnnotator - treeannotator. You need to alter the script in the 'bin' folder of the BEAST distribution, as I've done in the source code (the file 'treeannotator').
  • MUSCLE - muscle
  • MAFFT - mafft
  • Clustal-Omega - clustalo
  • PRANK - prank
  • MetAl - metal
  • Phlyomatic - phylomatic
