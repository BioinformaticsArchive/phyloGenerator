phyloGenerator automated phylogeny generation for ecologists

Downloads

phyloGenerator is simply a Python script, that can either be run from the command line, or loaded as a module to aid you in writing scripts to generate a phylogeny.

You can download a zipped up copy of the the package here. For those who are interested, you can also fork the project straight from GitHub.

Make sure you follow the download instructions carefully; installation is not hard but requires patience... I am writing a script to try and speed all this up!

Quickstart

  1. Python (>= 2.6.x, not 3.x)
  2. BioPython (>= 1.58 )
  3. Any phylogenetics software you want to use with it
  4. Download it
  5. ...import the script, and you're away!'
