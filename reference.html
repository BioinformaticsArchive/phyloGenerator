Reference

phyloGenerator's internals, and a guide to setting up configuration files'

Overview

phyloGenerator is really just a big Python script, and you can use its internals in your own scripts if you'd like. I haven't written a proper guide yet, but for the time being just looking at the internals of the script (ignore the classes, as they're used in running the actual script).

You can also make configuration files that will automate even more of phyloGenerator's features, allowing you to make phylogenies even more easily. However, this is only half-completed - but when it's finished, you'll see the details here!
